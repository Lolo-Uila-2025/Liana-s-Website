Two things I did With Lolo Uila That I enjoyed

The first thing that we did which I enjoyed doing was being introduced to what an Arduino Board is. A simple definition of what an arduino board is, a small single-board microcontroller inexpensive computer that can be used to interact with the physical world. My favorite thing that we did using our arduino boards was the security buzzer test. We used an altrosonic sensor to be able to sense when there is movement. We also used codes to make an LED light bulb go off whenever the sensor would go off. Another thing that we used almost every class was a bread board. We would used wires to connect our arduino boards to our breadboard. A breadboard is a board that is used to connect eletronic components to conduct various experiments and projects, like our sensor experiment. These are two things that we did in Lolo Uila this year.
